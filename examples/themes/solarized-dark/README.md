# solarized-dark

See [examples/themes/README.md](../README.md) for the full description and
comparison table.

## Build

From the repository root:

```bash
matisse build \
  -i examples/themes/talk.md \
  -o examples/themes/solarized-dark/out/ \
  --theme solarized-dark
```

Open `examples/themes/solarized-dark/out/index.html` in a browser.
