# dracula

See [examples/themes/README.md](../README.md) for the full description and
comparison table.

## Build

From the repository root:

```bash
matisse build \
  -i examples/themes/talk.md \
  -o examples/themes/dracula/out/ \
  --theme dracula
```

Open `examples/themes/dracula/out/index.html` in a browser.
