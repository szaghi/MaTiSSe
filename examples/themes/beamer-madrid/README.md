# beamer-madrid

See [examples/themes/README.md](../README.md) for the full description and
comparison table.

## Build

From the repository root:

```bash
matisse build \
  -i examples/themes/talk.md \
  -o examples/themes/beamer-madrid/out/ \
  --theme beamer-madrid
```

Open `examples/themes/beamer-madrid/out/index.html` in a browser.
